The "Profile Plots" graph plot the depth profiles for one parameter for each site selected. GLKN only measured depth profiles for water temperature, pH, specific conductivity, dissolved oxygen (mg/l), and dissolved oxygen (%). Each row of the figure shows the profile plot for each month the parameter was measured. Each column of the figure shows the years selected in the section above. The "Table" tab shows a screen-reader-friendly version of the data used to create the "Time Series" graph. The "Exceedances" tab shows the data associated with any values that exceed the threshold values for the parameter of interest.

Water Quality Thresholds

Some water quality parameters have thresholds that indicate "good" water quality. Checking the "Threshold" box will display this threshold as a line on the graph.

Downloads

You can download the graph in .png format by mousing over the graph and then clicking the camera icon. For each table, you can download a .csv or .xlsx version of the data.

Graphics Options

Clicking the Graphics Options button will allow you to adjust the graph height and width, the point and line color and size, and the font size.
