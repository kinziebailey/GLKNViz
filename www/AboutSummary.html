The "Summary" table generates summary statistics for water quality measurements. The "Summary" table uses the mean of the site visit's measurements when more than one measurement was taken for a parameter on one site visit.

The "Compare by" control allows you to group measurements to compare among groups.

Contents of "Summary" table:

  • Minimum: The lowest recorded value in the selected data.
  • Q1: 25th percentile; the value below which 25% of the data falls under.
  • Mean: The average of all the data values.
  • Median: 50th percentile; the central value in a dataset when sorted.
  • Q3: 75th percentile; the value below which 75% of the data falls under.
  • Maximum: The highest recorded value in the selected data.
  • Standard Deviation: Measure of the spread or variability of the data. A higher standard deviation indicates more variation from the mean, while a lower value suggests the data points are closer to the mean.
  • Site Visits: The total number of unique site visits.
  • Total Measurements: The total number of measurements for a given site.
  • Missing Values: The number of values not collected or reported.
