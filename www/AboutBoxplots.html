The "Boxplot" graph generates summary statistics (e.g. min, median, max) for the parameter of interest at each site selected. The user can selected how they want the summary statistics grouped (year, month, site). The "Table" tab shows a screen-reader-friendly version of the data used to create the "Time Series" graph. The "Exceedances" tab shows the data associated with any values that exceed the threshold values for the parameter of interest.

  • Year: Displays a different boxplot for each year. All data for each year will be included it the boxplot.
  • Month: Displays a different boxplot for each month. Data from all years contribute to each month's boxplot.
  • Site: Displays a different boxplot for each site in the selected park. All data collected at each site contributes to each boxplot.

Water Quality Thresholds

Some water quality parameters have thresholds that indicate "good" water quality. Checking the "Show Water Quality Threshold Line" box will display this threshold as a line on the graph.

Downloads

You can download the graph in .png format by mousing over the graph and then clicking the camera icon. For each table, you can download a .csv or .xlsx version of the data.
