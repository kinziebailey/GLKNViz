The "Time Series" graph plots one parameter against time for each site selected. For the parameters that have depth profiles (e.g. water temperature, pH, specific conductivity, DO (mg/L), DO (%)), only data in the upper 2 meters of the water column are shown. If there is more than one value, the following rules are applied.

  1. The mean is calculated for two values
  2. The median is calculated for three or more values

The "Table" tab shows a screen-reader-friendly version of the data used to create the "Time Series" graph. The "Exceedances" tab shows the data associated with any values that exceed the threshold values for the parameter of interest.

Water Quality Thresholds

Some water quality parameters have thresholds that indicate "good" water quality. Checking the "Threshold" box will display this threshold as a line on the graph.

Downloads

You can download the graph in .png format by mousing over the graph and then clicking the camera icon. For each table, you can download a .csv or .xlsx version of the data.
