The "Boxplot" graph generates summary statistics (e.g., median, max, min) and represents those summaries visually as boxplots. The "Summary" table shows a screen-reader-friendly version of the same information present in the "Boxplot" graph.

The "Compare by" control allows you to group measurements to compare among groups.

When more than one measurement was taken for a parameter on one site visit, the "Boxplot" graph uses the mean of those measurements.

  • Year: Displays a different boxplot for each year. All data for each year will be included it the boxplot.
  • Month: Displays a different boxplot for each month. Data from all years contribute to each month's boxplot.
  • Site: Displays a different boxplot for each stream in the selected park. In some parks monitoring takes place in only a single stream, so in those cases only a single boxplot is displayed.

Water Quality Thresholds

Some water quality parameters have thresholds that indicate "good" water quality. Checking the "Show Water Quality Threshold Line" box will display this threshold as a line on the graph.

Downloads

You can download the graph in .png format by mousing over the graph and then clicking the camera icon.

Graphics Options

Clicking the Graphics Options button will allow you to adjust the graph height and width, the point and line color and size, and the font size.
